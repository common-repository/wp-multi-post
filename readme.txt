== Description ==
WP Multi Post is a wordpress plugin, which makes your blogging experience easier and faster. Now you can post multiple articles at a time.

https://www.youtube.com/watch?v=RxtBNFfbO7I

== Installation ==
1. Upload `wp-multi-post` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==
= How many posts we can create at a time? =
Post limit: Up to fifteen(15) posts.
